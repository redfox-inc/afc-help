ユーザーの操作方法¶

ログイン¶

PCとスマートフォンでのログインはそれぞれ以下のようになります。

ログイン
PC スマートフォン
https://gps-punch.jp/gps-admin/ https://gps-punch.jp/transit-fare/

GPS Punch!のアプリと同じID、パスワードでログインします。


メイン画面¶

上部のメニューから交通費入力画面に移動します。


交通費の修正・確定をする¶

左上のカレンダーを押すと、1日の交通費金額入りのカレンダーが表示されます。

  • カレンダー内の色は以下の状態を表しています。
カレンダーの色について
色 状態
白 移動なし
グレー 移動あり、交通費未確定
緑 移動あり、交通費確定済み

操作画面¶

日付をクリックすると、その日の移動が地図上で見ることができ、交通費を修正・確定することができます。

  1. 現在選択している月のデータをCSVファイルで出力します。
  2. 現在選択している日の移動が一覧で表示されます。クリックすると地図上で出発地点、到着地点が確認できます。
  3. 現在選択している日の移動の修正、確定を行います。

画面右のコントロールパネルの機能は以下のようになっています。

  1. 個人設定を変更します。 ->詳細はこちら
  2. 日付を1日ずつ切り替えます。
  3. 同日内の 移動 を切り替えます。
  4. 新しい 移動 を追加します。
  5. 現在表示されている移動の内容を修正できます。 ->詳細はこちら
  6. 現在表示されている 移動 を削除できます。
  7. 経路検索結果の中から、別の経路候補に切り替えることができます。
  8. 現在選択している候補の路線情報が表示されます。
  9. 現在表示されている候補を選択します。
  10. 現在選択している日付のすべての 移動 の経路、金額を確定します。
確定した日付は修正できなくなります。

修正項目について
項目 説明
移動（電車、電車以外） 実際はタクシーで移動した場合などは、「電車以外」を選択します。
出発、行先 位置情報から検出した付近の駅名が表示されます。
※変更した場合は経路の再検索が必要です。
運賃 ICカードを選択すると1円単位で運賃を算出します。
座席 特急の座席を選択して、運賃を変更できます。
※候補の中に特急を利用する経路が含まれている場合のみ表示されます。
備考 メモなど、フリー入力ができます。

タクシー移動の場合は以下の手順になります。

  1. 移動欄にて「電車以外」を選択します。
  2. 領収証などの写真があれば添付します。
  3. 移動内容と金額を入力します。

不要な移動を削除する場合は以下の手順になります。

  1. 同日内の 移動 を切り替え、不要な移動を選択します。
  2. ゴミ箱ボタンを押すと、確認ダイアログが出るので「はい」を押します。
  3. 同日内の 移動 をすべて確認した上で、「交通費を確定」を押します。
  • 確定をしないと削除が完了しません。

Warning

交通費の変更ができるのは過去3ヶ月分までです。ご注意ください。

個人設定をする¶

交通費自動計算の結果を、より自分の利用ケースに合わせて算出できるよう、ここで調整します。

  1. 経路の検索オプションを設定することができます。 ->詳細はこちら
  2. 定期情報の編集ができます。
  3. 定期情報を削除できます。
  4. 定期情報を追加できます。

定期情報の設定¶

定期情報を設定することで、交通費算出の際に定期区間を除外して計算します。
  1. 自宅最寄り駅と職場最寄り駅を入力して「経路を検索ボタン」を押すと設定画面が表示されます。
  2. 検索結果の候補駅が表示されます。適切な駅を選択してください。（変更した場合は経路の再検索が必要です。）
  3. タブをクリックすると、別の経路候補に切り替えることができます。
  4. 現在選択している候補の路線情報が表示されます。
  5. 「この候補を選択」をクリックすると、選択している候補に切り替わります。
  6. 定期券の利用開始日を選択します。（定期の適用期間が決定します。）
  7. 「この経路を定期区間として設定」ボタンを押すと、定期券情報が確定します。
経路検索オプションについて
項目 説明
並び順 経路計算を時間が短い順、金額が安い順、乗り換え回数が少ない順のいずれかに変更できます。
特急座席 いつも利用する特急座席タイプをデフォルトにできます。
運賃 IC、現金のいずれかをデフォルトにできます。
直行直帰 出勤・退勤時の位置情報が定期区間外の場合は、その位置から職場まで（出勤）または、その位置から自宅まで（退勤）の経路を追加します。

直行直帰の計算について¶

直行直帰の計算を有効にしている場合と無効にしている場合の1日の計算例です。

定期券区間: 新宿（自宅） - 東京（職場） ある1日の移動:

  1. 三鷹の客先付近で出勤 (ここでGPS Punch!の記録を開始）
  2. 中野に移動
  3. 川崎に移動して退勤 (ここでGPS Punch!の記録を停止）
自動計算例
直行直帰を計算する 直行直帰を計算しない
新宿 - 三鷹 220円 なし
三鷹 - 中野 170円 三鷹 - 中野 170円
中野 - 横浜 640円 中野 - 横浜 640円
横浜 - 新宿 550円 なし

ユーザーの操作方法 (スマートフォン)¶

交通費計算記録を修正・確定する¶

移動内容を確認する
スマートフォンでは日付毎に移動の一覧が表示されます。いずれかの日付をタップするとその日の移動内容が確認・修正できます。

たとえば、7月23日の記録を見ると、交通費合計金額510円となっています。 問題がなければこの画面で「確定する」を押して完了します。

「内容を確認したいとき」「修正が必要なとき」は、個別の移動の表示を押します。

移動の詳細を確認する

移動の表示をタップすると、その詳細が確認でき、経路を修正をすることもできます。画面をスクロールして、一番下のボタンで変更・確定をさせます。

移動経路の修正

経路の修正方法は以下のようになっています。

修正項目について
項目 説明
出発、行先 位置情報から検出した付近の駅名が表示されます。 | ※変更した場合は経路の再検索が必要です。
目的地 交通費移動の目的地として表示されます。
運賃 ICカードを選択すると1円単位で運賃を算出します。
座席 特急の座席を選択して、運賃を変更できます。 | ※候補の中に特急を利用する経路が含まれている場合のみ表示されます。
備考 メモなど、フリー入力ができます。
移動記録を追加する
自分で記録を追加するときは、ホーム画面、または特定の日の交通費一覧画面の下、「新しい移動を追加する」をクリックして入力します。

交通費計算記録を確定する
交通費計算が確認できたら、「確定する」を押して確定させます。

Warning

交通費の変更ができるのは過去3ヶ月分までです。ご注意ください。


交通費自動計算の設定をする¶

設定画面について
左上の「メニュー」アイコンをタップすると、設定などの各種メニューが開きます。交通費計算の実態に合わせて「経路検索設定」と「定期設定」をします。

経路検索条件の設定

「並び順」は経路の候補の順番、「特急座席」で主に利用する席の種別など、各種の設定を切り替えることができます。下へスクロールして、最後に「更新」を押します。

経路検索オプションについて
項目 説明
並び順 経路計算を到着が早い順、乗り換え回数が少ない順、金額が安い順のいずれかに変更できます。
特急座席 いつも利用する特急座席タイプをデフォルトにできます。
運賃 ICカード、現金のいずれかをデフォルトにできます。
直行直帰 記録開始・記録終了時に自宅最寄り駅までの移動を自動的に作成します。


定期設定

通勤定期券の設定をします。定期券の期間と経路を設定すると、期間内の交通費計算を再計算することができます。

  1. 自宅最寄り駅を出発に、職場最寄り駅を行き先に入力して検索ボタンを押します。
  2. 経路を選択して「追加する」ボタンを押します。
